# 🎓 Education Website – Exam Preparation Platform

## 🚀 Overview

A modern exam-preparation platform designed to help students prepare for competitive exams through a clean, focused, and intuitive digital experience.

The platform emphasizes clarity, motivation, and structured learning, enabling users to track their progress and stay consistent.
